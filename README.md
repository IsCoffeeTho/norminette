# fast Norminette
## Installation
```sh
$> make
```

## Usage
* Runs on the current folder:
```
$> norminette
```
* Runs on the given filename(s):
```
$> norminette <filename>.[c/h]
```
* Prevents stopping on various blocking errors:
```
$> norminette -d
```
* Output all the debug logging:
```
$> norminette -dd
```
